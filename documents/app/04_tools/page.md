# Tools トップページ (/tools)

## 概要

自作のミニツール群を集めたハブページです。ブラウザだけで動き、スマホやタブレットでも操作しやすい UI を優先しています。ポモドーロタイマーやQR生成、SVG→TSX 変換など、日常作業を素早くこなすためのユーティリティを収録しています。

## 主なツール

- **Pomodoro**: モバイル対応ポモドーロタイマー。左フローバーがリアルタイムに色で満ち、付箋/画像/YouTube/ミニタイマーをドラッグ配置可能。
- **Color Palette**: 好きな色を並べて即座にパレット化・コピー。
- **QR Generator**: 入力した URL から QR コードを生成しダウンロード。
- **SVG2TSX**: SVG 画像を React コンポーネント (TSX) に変換。
- **Business Mail Block**: テンプレート化したビジネスメールの差し替え支援。
- **その他**: テキストカウンターや ae-expression 一覧など制作系の補助ツール。

## 使い方

- ツールカードをタップ/クリックして各ページへ移動。
- ほとんどのツールは入力内容をローカルに保存し、再訪時に復元します。
- スマホ・タブレットではタッチ操作でドロワーやドラックができるように調整済みです。
