# About トップページ

## 目的

著者の概要を紹介し、各サブページへの入口となるハブページ。

## 主な要素

- ヒーローヘッダー (名前、キャッチコピー)
- ナビゲーションカード (プロフィール / 名刺 / リンクマップ / 依頼ページ)
- 最新のポートフォリオやツールへのハイライトリンク

- `ContentItem` type: `profile` の概要が中心
