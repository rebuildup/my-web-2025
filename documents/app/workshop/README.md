# /workshop ルート仕様

プラグイン配布、ダウンロード素材、ブログ記事などを扱うクリエイティブハブ。

| パス                  | 役割                 |
| --------------------- | -------------------- |
| `/workshop`           | ハブ (カテゴリ選択)  |
| `/workshop/downloads` | 素材ダウンロード一覧 |
| `/workshop/plugins`   | プラグイン一覧       |
| `/workshop/blog`      | ブログトップ         |

ページごとの詳細は `page.md` を参照。
