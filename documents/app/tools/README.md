# /tools ルート仕様

オンラインユーティリティツールを提供するページ群。

| パス                   | ツール                     | 備考             |
| ---------------------- | -------------------------- | ---------------- |
| `/tools`               | ツール一覧                 | トップページ     |
| `/tools/color-palette` | カラーパレットジェネレータ | HEX/RGB/HSL 変換 |
| `/tools/estimate`      | 映像依頼料金計算機         | 計算ロジック付き |
| `/tools/qr-generator`  | QR コード生成              | vCard/URL 対応   |

各ツールの仕様は該当フォルダの `page.md` に記載します。
