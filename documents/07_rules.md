# 07_rules — 運用とコンプライアンスの要点

- 著作権/ライセンス: 外部ライブラリは各ライセンスを遵守。商用不可素材を同梱しない。
- 個人情報: 取得しない設計。問い合わせ時はメールアドレスとメッセージのみ想定。保存せず送信のみが原則。
- ログ: pm2 / アプリログには個人特定情報を残さない。保持は最小限、不要になればローテ/削除。
- セキュリティヘッダー: HSTS, X-Content-Type-Options, X-Frame-Options (DENY), CSP はリバースプロキシで設定。
- 秘密情報: `.env*` はコミット禁止。GitHub Secrets は定期ローテーション。
- メディア使用: Typekit (Adobe Fonts) の利用規約を遵守し、kit ID は公開情報として扱う。
