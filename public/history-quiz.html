資本主義経済の成立と発展

資本主義経済は、1760年頃から1830年頃にかけてイギリスで展開されたを経て成立した.ここに、生産手段を所有すると労働力を所有するの二大階級の成立を見ることができる.

資本主義経済 (⇔経済) の特徴は以下の通り.
① 生産手段の的所有 (⇔生産手段の的所有)
② 自由な生産活動 ( ⇔ ) 市場原理、契約自由の原則、主義
③ 商品生産社会･･･労働力も商品化
④ 追求社会

初期の資本主義は、政府の経済への介入を極力控える主義 ( ・ ) を基本理念として掲げており、この18世紀後半から19世紀末までの資本主義は資本主義と呼ばれる.しかし、こうした自由競争は、資本の集積・集中を通じて巨大企業の成立を促し、結果として市場の化が進んだ.この19世紀末から20世紀初にかけての資本主義は、資本主義と呼ばれる.

1929年にアメリカに端を発したは、主義を理念に掲げる資本主義経済を大きく変容させた.1933年にアメリカ大統領に就任したによる政策は、従来の主義を放棄し、政府の経済への積極的を試みた先駆的事例である.

これは不況下で不足している (実際に貨幣の支出を伴う需要) を政府がを拡大して創出しようとする政策であった.このように政府が経済にし (「」と呼ばれる)、民間部門と公的部門とが並存する経済は、と呼ばれる.また、経済活動の自由を一部制限して、経済的な平等を出来るだけ実現しようとする資本主義は、資本主義と呼ばれる.

1970年代以降、を背景にスタグフレーションが発生すると、管理政策の限界性が指摘されるようになった.政府の役割をし、規制緩和や減税政策を推進し、市場の自由競争に委ねる「」への転換を進めるべきとの声が強くなった.このような資本主義は、主義と呼ばれる.

主義の立場から経済再建を目指した政策として、アメリカの大統領によるレーガノミックス、イギリスの首相によるサッチャリズムが挙げられる.1980年代半ばに日本で行われた政権による三公社 (電電公社、専売公社、国鉄) のもこうした動きの例である.

社会主義経済の成立と市場経済への移行

19世紀半ばから資本主義の発展に伴って顕在化したの差の拡大やの発生 ( 問題) など資本主義経済の矛盾を生産手段の的所有による解消を目指す思想が広がった. 思想を集大成したのは、ドイツの経済学者 とであった.彼らは『』 (1848年) を公刊し、土地や工場、銀行など経済の重要な要素を公有化するための社会革命 (資本主義からへの移行) が必要であると説いた.または『』を著して、資本主義においては、は自由な立場で労働してもその報酬を完全には受け取れない仕組み ( ) があり、他方でもを無限に拡大することはできないため、こうした無理のある経済体制はいずれ崩壊すると論じた.

1917年にロシアではの指導のもと社会主義革命が展開され ( )、1922年にソ連 ( ) が成立した.1928年からは5年を一区切りとする「」が開始され、ソ連経済は順調に発展した.しかし、第二次世界大戦後しばらくすると資本主義国との経済格差が目立ち、国民経済も停滞した.政権下で進められたなど各種の政策もこうした事態を打開することはできず、1989年には東欧諸国の党政権が崩壊し、1991年にはソ連も解体され、その後化が進んだ.

中国では、社会主義経済にシステムを導入し、党と国家によるを基礎に行政指導で経済調節を行うが導入されている. またベトナムでは、1986年以来「 (刷新)」政策の名の下に、集権的な管理体制を是正し、システムや開放政策を導入している.今日では、社会主義経済を採用している国は、朝鮮民主主義人民共和国 (北朝鮮) やキューバなど少数に過ぎない.

和算の成立と社会背景

近代以前、特に時代における日本の数学のことをと呼ぶ.和算は、近世日本（時代）において、独自の発展を遂げる.

17世紀に入り幕府という強大な統一権力が出現し、全国各地の戦国大名の行動は統制される.大名は武力で領土拡張をはかることは不可能となり、自領の力を高めることによってのみ成長が可能となる状況に追い込まれる.が活発化し、全国各地で大規模な、干拓、開拓が起こり、堤防の築造やなどの高度な技術が要求された.

戦国時代末期になると、日本に新しい計算道具であるがから流入することでの基礎が確立する.の日本への導入の詳細は明らかではないが、16世紀末に盛んとなっていた日中貿易、東南アジア貿易に従事した商人らが導入したものと予想される.江戸時代特有の経済システムであるが展開することで、税務としてのの割り当て、 の換金といった業務を専門とする算術家の需要を全国に高め、が用いられた.

(1598-1672)は1627年にそろばんの教科書で当時の初等算術をも包含した『』(初版)を著す.江戸時代の人々の圧倒的な支持を受け、近世を通じて算術書の代名詞的存在となる.

現存する最古の初等算術書としては、が1622年頃に著した『』の存在が知られる.そろばんによる割算の仕方を述べ、面積や体積計算など日用算術にも及んでいる.一方『』は、内容が洗練され、田畑の面積計算や容器の体積計算、木の高さの測量といった的問題に加え、ねずみ算や継子立といった的問題を含む構成であり、人々の興味関心を惹きつける工夫が盛り込まれていた.内容は完全に吉田自身のオリジナルではなく、明代の数学書で程大位の『算法統宗』(1592年)の影響がうかがえる.
